# Icons

Icon component, implementing [Fontawesome](https://fontawesome.com/icons?d=gallery&s=solid&m=free).

* to add more icons use `~/configs/icons.js`
* you can pick from https://fontawesome.com/icons?d=gallery&s=solid&m=free
* for sizing and color use tailwind utility classes
